1.ftp service

Télécharger et partager des fichiers

Nous utilisons docker dans le serveur distant pour construire un serveur ftp. Afin de gérer votre ftp plus commodément, nous vous recommandons d'utiliser FileZila.

installer FileZila sous ubuntu

  • $sudo apt-get update
  • $sudo apt-get install filezilla

connecter au serveur

Pour démarrer Filezila ,vous pouvez taper filezila dans terminal et vous allez voir comme figure 1

Remplissez ensuite host ci-dessus :

  • host :52.47.87.250
  • Username :user
  • Password :userpwd
  • Port :21

figure 1

figure 2

Parcourir / modifier les fichiers

FileZilla fournit des fonctions de navigation et de modification, qui doivent être modifiées dans Edition-> Paramètres-> Edition de fichiers-> Associations de types de fichiers, qui correspondent au type de fichier et au programme d'accès correspondant utilisé. Si vous souhaitez accéder aux fichiers txt / fichiers c ++, nous vous recommandons d'utiliser Geany, doc utilise libreoffice

Installez le programme correspondant

  • $sudo apt-get install geany
  • $sudo apt-get install libreoffice
  • Pour le type de pdf on peut utiliser document Viewer(déjà installé sous ubuntu),sinon $sudo apt-get install okular

Ensuite, nous devons remplir le chemin correspondant, utiliser which 'application' dans le terminal pour trouver le chemin correspondant comme la figure 3
